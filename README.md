# Node Rest API
## Installation

```bash
$ npm install
```

## Running the app

```bash
# setup development
$ npm run setup:dev
# run development
$ npm run start

# watch mode - http://localhost:3000/api/
$ npm run start:dev

# setup production mode
$ npm run setup:prod
# run production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
